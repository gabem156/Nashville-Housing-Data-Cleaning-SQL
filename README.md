# Nashville Housing Data Cleaning

## 📊 Project Overview
This project focuses on cleaning and preparing a raw Nashville housing dataset using SQL. The goal is to transform messy, inconsistent data into a structured and reliable dataset that can be used for analysis and visualization.

The dataset includes property sales data such as addresses, sale dates, prices, and ownership details.

---

## 🧹 Data Cleaning Objectives
In this project, I performed the following cleaning tasks:

- Standardized date formats for consistency
- Filled or handled missing values where appropriate
- Removed duplicate records
- Split and cleaned address fields (e.g., property address vs. owner address)
- Converted inconsistent text values (e.g., "Y/N" → "Yes/No")
- Removed unnecessary columns
- Fixed formatting issues in key fields like city, state, and zip codes
- Ensured data integrity for analysis
